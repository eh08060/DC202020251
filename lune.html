git
세이브 포인트 생성, 관리 프로그램
복원 위치 생성관리 프로그램
자동은 아니고, 수동. 대신 내가 지점을 정할 수 있다.
클라우드 서비스는 자동인데, 시점은 대체로 아무 때나.
github
생성, 관리된 것을 온라인에 올리고 내릴 수 있게 해주는 서비스.
유사품 있습니다. gitbucket 등.
git 주요 명령어
commit : 세이브 포인트 생성
파일 - 변화한 파일 - 스테이지에 올린 파일
commit은 스테이지에 올린 파일만 세이브 포인트에 반영.
스테이지에 안 올린 파일은 변화한 파일로만 남아 있고, 세이브 포인트에 반영 안 됨.
push : 원격지에 업로드
pull : 원격지에서 다운로드
fetch :
